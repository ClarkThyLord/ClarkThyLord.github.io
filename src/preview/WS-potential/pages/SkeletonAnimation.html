WS Potential

WS Potential is for the newbies, hobbyist, and the professionals. Removing all the complicated and ridiculous stuff from them, and implementing all of their shining features while improving upon them. Most image manipulation softwares don’t have game developers in mind, I don’t blame that for that; but, unlike them I do which is why this software will be suitable for any game developer having support for scripts, particles, physics, and more. Their is no fee for using WS Potential, and all it’s code is open source so if you’re looking to fix or improve something you can do so!
